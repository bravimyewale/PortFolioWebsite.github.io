SafeNest – Smart Child Emergency Call and Geofencing Alert System using ESP32


SafeNest is an IoT-based child safety device designed to ensure real-time monitoring and emergency communication. The system uses ESP32, GSM, and GPS modules to track live location, send SOS alerts, and create geofencing zones that trigger automatic notifications when breached.

Technologies Used: ESP32, GSM Module, GPS Module, Arduino IDE


Key Highlights:

Implemented live SMS alerts with Google Maps links and automatic voice calls during emergencies.

Achieved ±5 m GPS accuracy and 5–8 s SMS response time.

Works without Internet dependency, ideal for child, elderly, and personal safety.

ESP32-Based Voice Assistant Using Gemini AI


A real-time embedded voice assistant integrating Google Cloud Speech-to-Text and Gemini AI APIs to enable intelligent voice-based interaction. The system performs voice recognition, processing, and response generation using the ESP32 microcontroller.

Technologies Used: ESP32, INMP441 Microphone, MAX98357A Amplifier, Google Cloud APIs, Python, FreeRTOS


Key Highlights:

Implemented I²S audio communication for high-quality digital voice processing.

Achieved 83.3% speech recognition accuracy with efficient multitasking using FreeRTOS.

Enables low-cost, scalable IoT voice assistant functionality.

Gesture-to-Servo Command Mapping Using Optical Flow and FIR Filtering in Simulink


This project simulates in MATLAB Simulink using optical flow–based motion detection and servo motor modeling. Dynamic FIR filters and a gesture decision algorithm classify hand movements into directional controls for the arm. The system achieves 80–85% accuracy, offering a reliable, cost-effective, and sensorless solution for assistive and automation applications.

Technologies Used: MATLAB Simulink, Image Processing Toolbox, Control System Toolbox, FIR Filter Design


Key Highlights:

Implemented optical flow and ROI-based gesture recognition with 80-85% accuracy.

Developed FIR filtering for real-time noise reduction and smooth servo response and Modeled servo motor dynamics using first-order lag representation for realistic simulation..

Demonstrated a cost-effective, sensorless gesture control approach suitable for assistive robotics and automation.

Electronic Dice using IC 555 and IC 4017


A digital dice circuit designed using IC 555 as an astable multivibrator and IC 4017 as a decade counter to simulate a traditional dice roll. The circuit sequentially lights six LEDs when a push button is pressed and stops randomly upon release, displaying the final dice value.

Technologies Used: IC 555 Timer, IC 4017 Decade Counter, LEDs, Resistors, Capacitor, Push Button.


Key Highlights:

Implemented IC 555 timer in astable mode to generate clock pulses for LED sequencing.

Integrated IC 4017 counter for controlling six LED outputs representing dice numbers (1–6).

Adjustable speed control using a 10k variable resistor for realistic dice rolling effect.

Compact, low-cost circuit demonstrating timer–counter interfacing and digital logic design.

Smart Street Light System Using Aurdino Nano


A microcontroller-based smart street lighting system developed using Arduino Nano and IR sensors for automatic street light control. The system detects vehicles or objects entering the street and sequentially turns ON the LEDs with time delay, switching them OFF as the object moves ahead — minimizing power wastage and improving efficiency.

Technologies Used: Arduino Nano, IR Sensors, LEDs, Embedded C Programming.


Key Highlights:

Implemented sequential lighting control with IR object detection for real-time automation.

Designed delay-based LED activation and deactivation for efficient energy usage.

Demonstrated microcontroller-based control logic suitable for smart city applications.

Achieved reduced power consumption compared to conventional static street lighting systems.
